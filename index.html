Projects

Projects

FFA | Fit fir den Aarbechtsmaart

Description

This project was about creating a website for my school. My school needed the website to allow students to enroll in company interviews and company presentations. And everything should be managed via an admin panel.

Implementation

The project was at the beginning a competition among other students for a prize. My group of three people won the competition.

After the competition was finished, I got a request to develop the website further and finally publish it. The use case and functions of the website have expanded dramatically in the meantime. To determine these functions, there were regular meetings with the client.

I worked on this project for over 500 hours.

Software

During the project I worked on the Ubuntu 20.04 LTS operating system, the website was then hosted on an Apache2 web server. I programmed the website with Visual Studio code, and automatically uploaded it to the Apache2 folder with an SFTP plugin. The back-end ran via PHP and the front-end via Javascript.
